Programación Front-End

El desarrollo de una web o de una app web puede dividirse en dos: programación Front-End y programación Back-End. Ambas son llevadas a cabo por un programador.

El Front-End es la parte visual, la interfaz, y se encuentra en la parte del cliente. En cambio, el Back-End es la parte que no vemos, la que se encuentra en la parte del servidor y procesa la información recibida a través del frontend. De cada uno de estos aspectos se encargan la programadora Frontend y la programadora Backend, pero también existe un tercer tipo, la programadora Fullstack, que es la que maneja ambas programaciones.

Entonces, la programación Front-End es la encargado de desarrollar la parte visual de la web: el contenido, su estructura y estilo (la tipografía, la colorimetría...) Su labor es realmente importante ya que para que una página web funcione tiene que tener un diseño atractivo e intuitivo para el receptor.

Lenguajes Front-End

HTML, CSS y Javascript son los lenguajes principales del Frontend, de los que se desprenden una cantidad de frameworks y librerías que expanden sus capacidades para crear cualquier tipo de interfaces de usuarios. React, Redux, Angular, Bootstrap, Foundation, LESS, Sass, Stylus y PostCSS son algunos de ellos.

Estos lenguajes, aunque están orientados al front-end, no realizan las mismas funciones dentro de la programación. A continuación tenemos las principales características y de cada uno:

  • HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de las siglas que corresponden a HyperText Markup Language, es decir, lenguaje de Marcas de Hipertexto. Este lenguaje se usa para crear el contenido de nuestra web.
  • CSS es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado. Sus siglas corresponden a Cascading Style Sheets, hojas de estilo en cascada.
  • Javascript, abreviado (JS) es un lenguaje estándar de programación web para la interacción y automatización de las páginas web y aplicaciones web. Es decir, se encarga de la funcionalidad de la página.
  • Lenguajes compilados: estos lenguajes también son de utilidad para el desarrollo Front-End ya que ayudan realizar un código más limpio y fácilmente manipulable que luego se traduce a los lenguajes que leen los navegadores como HTML. Algunos ejemplos son Sass o SCSS (ambos compiladores de CSS) y PUG (compilador de HTML). Durante el curso también aprenderemos a usar Sass para mejorar nuestro código.
